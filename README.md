Compilation of projects completed for a Marketing Analytics course for the MS in Applied Data Science program at Syracuse University

Course Description: This course will focus on developing marketing strategies and resource allocation decisions driven by quantitative analysis. Topics covered include market segmentation, market response models, customer profitability, product recommendation systems, churn predictions, media attribution models, and resource allocation.
